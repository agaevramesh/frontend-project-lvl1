### How to install:
```
$ make install
$ make publish
```

### There are 5 games:
1. brain-even
Answer number even or not

2. brain-gcd
Find greatest common divisor of two numbers

3. brain-progression
Find number is missing in the progression

4. brain-calc
Find result of the expression

5. brain-prime
Answer number prime or not

### To start game:
```
$ brain-even
```
